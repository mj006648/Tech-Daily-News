# Tech-Daily-News 🚀

Technical Trend Monitoring & Bilingual Archive System

## 📌 Project Overview
이 프로젝트는 AI, 클라우드 인프라, 하드웨어 및 오픈소스 생태계의 최신 동향을 매일 아침 자동으로 수집하고 아카이빙하는 **지능형 기술 모니터링 시스템**입니다. 전 세계의 저명한 기술 블로그와 RSS 피드를 분석하여 핵심 정보를 요약하고 한국어로 번역하여 제공합니다.

## 🛠 How It Works (Automation)
이 시스템은 **GitHub Actions**를 기반으로 100% 자동화되어 작동합니다.
1.  **Schedule**: 매일 오전 09:00 (KST)에 GitHub 서버에서 워크플로우가 자동으로 트리거됩니다.
2.  **Fetch**: OpenAI, CNCF, NVIDIA 등 사전 정의된 고품질 기술 소스(RSS)로부터 지난 24시간 이내의 기사를 수집합니다.
3.  **Process**: 수집된 영문 기사를 분석하여 핵심 요약을 추출하고, `googletrans`를 통해 한국어로 실시간 번역합니다.
4.  **Archive**: 생성된 리포트를 `archive/YYYY-MM-DD/` 경로에 저장하고 메인 페이지의 인덱스를 업데이트합니다.
5.  **Commit**: 모든 변경 사항을 GitHub Action Bot이 자동으로 커밋하고 푸시합니다.
